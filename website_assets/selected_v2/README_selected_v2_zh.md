# Website Assets Selected v2

## 用途

這個資料夾存放 `website_visual_prototype_v2.html` 使用的 18 張正式候選圖。所有圖片均由遊戲 app 內部 asset 直接複製，避免網站只依賴臨時截圖。

## 選圖原則

1. **先建立遊戲世界感**：hero 與大型段落使用地圖、城市、改革場景和 cinematic asset。
2. **再說明遊戲機制**：用港口、船政、同文館、鐵路、電報、輪船等圖像支撐多地點探索、史料判讀、限制與後果。
3. **保留研究邊界**：三個尚未有理想截圖的關鍵位先預留 placeholder，不用低質截圖硬填。
